[Via 轻插件](https://viayoo.com/zh-cn/)
======

`Via 轻插件`是专门为 Via 浏览器定制的 JavaScript 插件

### Via 轻插件能做什么
可以干掉广告，美化网页，让浏览体验更佳。
最关键是，你不用为了仅仅使用一小部分的功能，而下载 1 到 2 个 G 的 APP

### 其他浏览器可以使用吗
可以，请自行下载脚本，复制添加到浏览器

### 调试插件
手机端调试可以使用 [vConsole](/script/手机网页调试.js)
[官方地址](https://github.com/Tencent/vConsole)
